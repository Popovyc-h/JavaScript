## Завдання 1. Обробка даних форми реєстрації (Рядки)

При роботі з користувацькими даними завжди потрібна валідація та нормалізація інформації перед відправкою на сервер.

```javascript
const formData = {
    username: '  UserName123  ',
    email: '  USER@EXAMPLE.COM  ',
    phone: '+380 (50) 123-45-67',
    fullName: 'іван  петрович   сидоренко',
}
```

**Що потрібно зробити:**

1. Створити функцію `normalizeUsername(username)`, яка приймає ім'я користувача і повертає його у нормалізованому вигляді:
   - Видалити пробіли з початку та кінця (використати метод рядків).
   - Привести до нижнього регістру.
   - Якщо довжина імені менше 3 символів або більше 20 — повернути `null`.
   - Якщо в імені є символи, окрім літер латиниці та цифр — повернути `null`.

```javascript
normalizeUsername('  UserName123  ') // "username123"
normalizeUsername('ab') // null (занадто коротке)
normalizeUsername('user@name') // null (недозволені символи)
```

> **Підказка:** використовуйте `toLowerCase()`, `trim()`, та цикл `for` для перевірки символів через `charCodeAt()` або прямого порівняння символів.

2. Створити функцію `normalizeEmail(email)`, яка приймає email і повертає його у нормалізованому вигляді:
   - Видалити пробіли з початку та кінця.
   - Привести до нижнього регістру.
   - Перевірити, чи є в рядку символ `@` (використати `indexOf` або `includes`).
   - Перевірити, чи є крапка після `@` (шукати крапку після позиції `@`).
   - Якщо валідація не пройдена — повернути `null`.

```javascript
normalizeEmail('  USER@EXAMPLE.COM  ') // "user@example.com"
normalizeEmail('notanemail') // null
normalizeEmail('test@') // null
```

3. Створити функцію `formatPhoneNumber(phone)`, яка приймає номер телефону і повертає його у форматі `+380501234567` (тільки цифри з кодом країни):
   - Видалити всі символи окрім цифр та `+` (пройтися циклом `for` по кожному символу).
   - Якщо номер починається з `380` (без `+`) — додати `+` на початку.
   - Якщо номер починається з `0` — замінити на `+380`.
   - Якщо після нормалізації довжина не дорівнює 13 символам (включно з `+`) — повернути `null`.

```javascript
formatPhoneNumber('+380 (50) 123-45-67') // "+380501234567"
formatPhoneNumber('0501234567') // "+380501234567"
formatPhoneNumber('501234567') // null
```

4. Створити функцію `capitalizeFullName(fullName)`, яка приймає повне ім'я і повертає його з великої літери для кожного слова:
   - Видалити зайві пробіли з початку, кінця та між словами (може бути кілька пробілів підряд).
   - Кожне слово має починатися з великої літери, решта — малі.
   - Повернути рядок, де слова розділені одним пробілом.

```javascript
capitalizeFullName('іван  петрович   сидоренко') // "Іван Петрович Сидоренко"
capitalizeFullName('  марія    ') // "Марія"
```

> **Підказка:** розбийте рядок на слова через цикл, шукаючи пробіли. Збирайте непробільні символи у слова, перетворюйте перший символ кожного слова на великий через `toUpperCase()`, решту — через `toLowerCase()`.

## Завдання 2. Кошик товарів в інтернет-магазині (Масиви, Числа)

В e-commerce проєктах завжди потрібно працювати з кошиком товарів: додавати, видаляти, рахувати суму, застосовувати знижки.

```javascript
const cart = [
    { id: 1, name: 'Ноутбук', price: 25000, quantity: 1 },
    { id: 2, name: 'Миша', price: 500, quantity: 2 },
    { id: 3, name: 'Клавіатура', price: 1200, quantity: 1 },
]
```

**Що потрібно зробити:**

1. Створити функцію `addToCart(cart, product)`, яка додає товар до кошика:
   - Якщо товар з таким `id` вже є — збільшити його `quantity` на значення з нового товару.
   - Якщо товару немає — додати його в кінець масиву (використати `push`).
   - Функція має повертати **новий** масив (скопіювати всі елементи в новий масив через цикл `for`).

```javascript
const newCart = addToCart(cart, { id: 2, name: 'Миша', price: 500, quantity: 1 })
// товар з id=2 тепер має quantity: 3

const newCart2 = addToCart(cart, {
    id: 4,
    name: 'Монітор',
    price: 8000,
    quantity: 1,
})
// новий товар додано в кінець
```

2. Створити функцію `removeFromCart(cart, productId)`, яка видаляє товар з кошика за `id`:
   - Пройтися циклом `for` по масиву, знайти індекс товару з потрібним `id`.
   - Створити новий масив без цього елемента (скопіювати всі елементи, окрім знайденого).
   - Якщо товар не знайдено — повернути копію оригінального масиву.

```javascript
const newCart = removeFromCart(cart, 2)
// товар "Миша" видалено з кошика
```

3. Створити функцію `calculateTotal(cart)`, яка рахує загальну суму кошика:
   - Пройтися циклом `for` по всіх товарах.
   - Для кожного товару порахувати `price * quantity`.
   - Повернути загальну суму, округлену до 2 знаків після коми (використати `toFixed`, але перетворити результат назад у число).

```javascript
calculateTotal(cart) // 27200
```

4. Створити функцію `applyDiscount(cart, discountPercent)`, яка застосовує знижку до всіх товарів:
   - Пройтися циклом `for` по масиву.
   - Для кожного товару зменшити ціну на `discountPercent` відсотків.
   - Округлити нову ціну до 2 знаків після коми (використати `toFixed` і перетворити назад).
   - Повернути **новий** масив з новими цінами (створити копії об'єктів вручну через цикл).

```javascript
const discountedCart = applyDiscount(cart, 10)
// всі ціни зменшились на 10%
// Ноутбук: 22500, Миша: 450, Клавіатура: 1080
```

5. Створити функцію `getMostExpensiveProduct(cart)`, яка повертає найдорожчий товар (за одиницю товару, не за загальну вартість):
   - Пройтися циклом `for` по масиву.
   - Знайти товар з найбільшим значенням `price`.
   - Повернути цей товар (сам об'єкт).

```javascript
getMostExpensiveProduct(cart) // { id: 1, name: "Ноутбук", price: 25000, quantity: 1 }
```

## Завдання 3. Генератор випадкових паролів та PIN-кодів (Числа, Рядки)

Для реєстрації користувачів або скидання пароля часто потрібно генерувати випадкові коди.

**Що потрібно зробити:**

1. Створити функцію `generatePIN(length)`, яка генерує PIN-код (тільки цифри) заданої довжини:
   - Використати `Math.random()` для генерації випадкових цифр від 0 до 9.
   - Згенерувати `length` цифр і зібрати їх у рядок через цикл `for`.
   - Повернути рядок.

```javascript
generatePIN(4) // "1234" (випадкові 4 цифри)
generatePIN(6) // "789012" (випадкові 6 цифр)
```

> **Підказка:** `Math.random()` повертає число від 0 до 1. Помножте на 10 і використайте `Math.floor()` для отримання цілого числа від 0 до 9.

2. Створити функцію `generatePassword(length, includeNumbers, includeSymbols)`, яка генерує випадковий пароль:
   - База символів: малі літери латиниці `'abcdefghijklmnopqrstuvwxyz'`.
   - Якщо `includeNumbers === true` — додати цифри `'0123456789'`.
   - Якщо `includeSymbols === true` — додати символи `'!@#$%^&*'`.
   - Згенерувати `length` випадкових символів з доступного набору (використати `Math.random()` і індекс рядка).
   - Повернути рядок пароля.

```javascript
generatePassword(8, true, false) // "a3f7k2m9" (літери + цифри)
generatePassword(10, true, true) // "a3!f7@k2m9" (літери + цифри + символи)
```

> **Підказка:** спочатку зберіть всі доступні символи в один рядок, потім у циклі `for` генеруйте випадковий індекс і беріть символ за цим індексом.

3. Створити функцію `validatePIN(pin)`, яка перевіряє чи є PIN-код валідним:
   - PIN має складатися **тільки** з цифр.
   - Довжина має бути рівно 4 або 6 символів.
   - Повернути `true` або `false`.

```javascript
validatePIN('1234') // true
validatePIN('123456') // true
validatePIN('12345') // false (довжина 5)
validatePIN('12a4') // false (є літера)
```

4. Створити функцію `generateSecureCode(prefix)`, яка генерує код у форматі `prefix-XXXX-XXXX-XXXX`, де `X` — випадкова цифра:
   - Використати `generatePIN(4)` тричі.
   - Зібрати рядок у форматі `prefix-XXXX-XXXX-XXXX`.

```javascript
generateSecureCode('PROMO') // "PROMO-1234-5678-9012"
generateSecureCode('GIFT') // "GIFT-4321-8765-0192"
```

5. Створити функцію `calculatePasswordStrength(password)`, яка повертає силу пароля (число від 0 до 100):
   - Базова сила = 0.
   - За кожну літеру (перевірити `charCodeAt()` або прямим порівнянням) — додати 2 бали.
   - За кожну цифру — додати 3 бали.
   - За кожен спецсимвол (`!@#$%^&*`) — додати 5 балів.
   - Якщо довжина >= 12 символів — додати 20 балів.
   - Максимальна сила = 100 (якщо результат більше 100 — повернути 100).

```javascript
calculatePasswordStrength('password') // 16 (8 літер × 2)
calculatePasswordStrength('pass123') // 23 (4 літери × 2 + 3 цифри × 3)
calculatePasswordStrength('P@ss123!Long') // 100 (літери + цифри + символи + довжина >= 12)
```

## Додаткові вимоги

1. Усі функції мають бути **чистими** (не модифікувати вхідні дані), окрім випадків, де це явно зазначено.
2. Не використовувати методи масивів (`map`, `reduce`, `filter`, `forEach`, `find`) — тільки цикли `for`, `for..in`, `for..of`.
3. Коментувати код не потрібно, але назви змінних мають бути змістовними.
4. Тестувати кожну функцію хоча б на 2-3 прикладах (можна просто викликати в кінці файлу і виводити результат у консоль).
